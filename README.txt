Welcome to Drupal Ticket, where you can use Drupal as a ticketing system.

INSTALLATION
============
To install this module, there are two ways that you can do this.

Option 1 (the basic way):
  1. Copy the module into your sites/all/modules/contrib folder.
  2. Navigate to admin/modules.
  3. Check the checkbox "Drupal Ticket" and select "Save Configuration".

Option 2 (the drush way):
  1. In your sites/all directory, type the command "drush dl dticket"
  2. Type "drush en dticket" to enable the module.

Once you completed one of the options, navigate to admin/config/dticket to
configure the module.
